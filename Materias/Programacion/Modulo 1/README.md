# 📁 Módulo 1: "The Core Logic" (Fundamentos del Pensamiento Estructurado)

> **"Code is like magic; you write words, and things happen. Pero antes de la magia, hay lógica — y esa la aprendes aquí primero, sin una sola línea de código todavía."**
> 
> Bienvenid@ al Módulo 1. Aquí es donde se rompe el hielo. No importa si nunca has programado nada en tu vida; en esta carpeta aprenderás a hablar el idioma de las computadoras. Empezamos sin sintaxis — lo que venimos a entrenar primero es tu **lógica**. Al terminar este bloque, podrás entender cómo piensa un software por dentro, y ya vas a tener tus primeras líneas de Python real corriendo.

### 🎯 Objetivos del Módulo

- **Pensar antes de programar:** Entender qué es un algoritmo y construir pseudocódigo, sin depender de ningún lenguaje todavía — la habilidad que de verdad separa a quien programa de quien solo copia scripts.
    
- Entender cómo la computadora almacena información usando **Variables y Tipos de Datos**, aterrizando por primera vez la lógica del módulo a código real de Python.
    
- Aprender a darle control a tus programas mediante condiciones (`if / elif / else`) para que tomen decisiones solos.
    
- Automatizar tareas repetitivas en milisegundos usando bucles (`for / while`).
    

### 🗺️ Lecciones clave

- **📄 Lección 1.1: Pensamiento Computacional (Qué es un Algoritmo):** Sin código todavía. Qué es un algoritmo, cómo escribir pseudocódigo, las 3 estructuras universales (secuencia, selección, iteración) que forman cualquier programa que existe, y diagramas de flujo.
    
- **📄 Lección 1.2: Variables y Tipos de Datos (De la Incógnita al Código Real):** Aquí aparece Python por primera vez. Cómo guardar información en variables, los 4 tipos de datos esenciales (texto, números, decimales, booleanos), y cómo convertir entre ellos.
    
- **📄 Lección 1.3: El Camino de las Decisiones (Condicionales):** Estructuras condicionales (`if/elif/else`), operadores de comparación y lógicos. Cómo hacer que tu código reaccione según la situación.
    
- **📄 Lección 1.4: El Poder de la Automatización (Bucles):** Bucles `for` y `while`. Cómo repetir instrucciones sin escribir código de más, y por qué un bucle mal pensado puede correr para siempre.
    

> **Nota:** las Funciones (crear bloques de código reutilizables) no se cubren en este módulo — ya las viste en **Matemáticas Módulo 1, Lección 1.4** ("Funciones como Máquinas de Código"), con la misma lógica de input → proceso → output aplicada primero a matemáticas. Las verás de nuevo, ahora en Python, en el Módulo 2 de esta materia.
